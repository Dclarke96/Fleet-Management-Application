Title: Vacation Scheduler

Purpose: The Vacation Scheduler is a mobile application developed for Android devices that allows users to manage personal vacations and associated excursions. The app provides functionality to create, edit, and delete vacation plans, set alerts, and view detailed information for each excursion and vacation entry.

How to Operate the Application:
Launch the App

When the app opens, you are greeted with a Home Screen containing a button to view vacations.

View Vacations

Tap “VIEW VACATIONS” to see a list of all saved vacations.

You can add a new vacation using the Add button.

Add/Edit a Vacation

Enter vacation details like title, destination, start/end dates.

Save to view it in the list.

Tap a vacation to open its detail view.

Edit or delete using available options.

View Excursions

In a vacation detail view, tap “View Excursions” to see all excursions for that vacation.

Add new excursions with title, description, and dates.

Each excursion includes optional alert functionality.

Alerts & Validation

Alerts are scheduled using Android’s alarm service.

Dates are validated to ensure excursions fall within the parent vacation date range.

APK Deployment Version:
The signed APK is built and tested on:

Android SDK Version: 33

Minimum SDK Version: 26 (Android 8.0 - Oreo)

Compatible with most Android 8.0+ devices.
